📋 Cara Hubungkan ke Google Sheets

Ikuti langkah berikut — hanya 5 menit!

1

Buka Google Sheets kamu

Buat sheet baru atau gunakan yang sudah ada. Pastikan kolom pertama baris 1 berisi header: Tanggal, Toko/Sumber, Item, Jumlah, Kategori, Tipe

2

Buka Apps Script

Di Google Sheets, klik menu Extensions → Apps Script. Hapus semua kode yang ada, lalu paste kode ini:

function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); sheet.appendRow([ data.tanggal, data.toko, data.item, data.jumlah, data.kategori, data.tipe ]); return ContentService.createTextOutput(JSON.stringify({status:"ok"})).setMimeType(ContentService.MimeType.JSON); }
3

Deploy sebagai Web App

Klik Deploy → New deployment. Pilih type Web app. Set "Execute as" → Me, dan "Who has access" → Anyone. Klik Deploy, lalu copy URL-nya.

4

Paste URL di FinBot

Tempel URL yang dicopy tadi ke kolom webhook di bagian atas aplikasi ini, lalu klik Simpan. Selesai! ✅
